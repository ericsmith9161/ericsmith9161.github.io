SZYSLAK

ABOUT THE PROJECT

A full stack single-page Slack clone where users can create and edit channels, add photos, message channels and direct message one or more pals. Created with Rails backend and React/Redux frontend with PostgreSQL database. Utilizes websockets, AWS S3, modals and dropdowns for functionality, and polymorphism as well as reusable components to keep code DRY.

Feel free to browse channels, or create one of your own. Once you're in a channel, you're live!


Send direct messages to your friends and associates, whether just one or many at a time

Technologies: Ruby, Ruby on Rails, React, Redux, HTML, CSS, Websocket, AWS S3
